# Healthcare Analytics: Chronic Disease Prediction

## 🚀 Project Overview
This project uses Machine Learning to predict whether a patient has a chronic disease based on factors like BMI, Glucose levels, and Physical Activity.

## 📱 Mobile-First Development
**Note:** This entire project was developed and deployed using a **Mobile Device** (Google Colab), showing that Data Science is possible anywhere.

## 📊 Performance
- **Accuracy:** 83.67%
- **Model:** Random Forest Classifier
